!carpeta!descripcion                                                                                                                                                                                                      !
!-------!-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------!
!/      !                                                                                                                                                                                                                 !
! boot  ! es la carpeta donde reside el sistema de Bootstrap (arranque ) del sistema                                                                                                                                      !
! dev   ! es la carpeta donde está devinidos los DEVICES, las periféricas de Input o Output, discos, CPU, tarjetas de red...                                                                                              !
! etc   ! es la carpeta de configuracion                                                                                                                                                                                  !
! home  ! es la carpeta de los usuarios                                                                                                                                                                                   !
! media ! es la carpeta en la cual se "montan" los discos externos (cd, Hd usb, pinchos..)                                                                                                                                !
! mnt   ! es otra carpeta que se utilizaba para el mismo motivo, allí, por ejemplo, se solían montar discos para aumentar el espacio de almacenamiento                                                                    !
! opt   ! en esta carpeta se suelen instalar thirdypart add-ons parte del Sistema operativo                                                                                                                               !
! proc  ! es una carpeta de sistema en la cual hay un filesystem virtual; reporta informaciones en tiempo real del estado del kernel y procesos activos.. ejemplo: cat /proc/meminfo os enseñará el estado de la memoria  !
! root  ! es la carpeta HOME de Root el superuser. sono Él puede acceder                                                                                                                                                  !
! run   ! es una carpeta en la cual se guardan los datos temporales de los programas/servicios en uso.                                                                                                                    !
! srv   ! vieja carpeta que se utilizaba para poner a disposición discos                                                                                                                                                  !
! sys   ! es un filesystem virtual de sistema. se utiliza para modificar el comportamiento del Hardware mientra se trabaja... Evidentemente no se tiene que tocar.                                                        !
! tmp   ! es una carpeta de files temporales                                                                                                                                                                              !
! usr   ! es una carpeta que contiene programas y datos del USUARIO. no es personal, allí van todos los programas que NO suelen ser de sistema.                                                                           !
! var   ! es la carpeta donde se hay los datos variables... los logs, las cache, los locks.. (/var/log por ejemplo..)                                                                                                     !
!-------!-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------!
